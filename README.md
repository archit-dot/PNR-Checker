# 🚆 PNR Status Checker

A simple Python application that checks the status of an Indian Railway PNR using the RapidAPI IRCTC API.

## Features

- Enter a PNR number
- Displays Train Name
- Displays Train Number
- Shows Source & Destination
- Shows Chart Status
- Displays Passenger Coach
- Displays Seat/Berth
- Displays Current Booking Status

## Technologies Used

- Python
- Requests Library
- RapidAPI
- IRCTC PNR API

## Installation

```bash
pip install requests
```

## Usage

1. Replace `YOUR_API_KEY` with your RapidAPI key.
2. Run:

```bash
python pnr_status.py
```

3. Enter your PNR number.
